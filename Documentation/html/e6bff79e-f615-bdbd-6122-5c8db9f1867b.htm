MyKanban
MyKanban is an open source project to allow developers to create Kanban boards and related applications using a structured object model. The MyKanban project provides all the structures for the database in MySql and SQL Server form, a complete .NET class library, and sample Windows, web and mobile applications demonstrating the use of the MyKanban object library.
