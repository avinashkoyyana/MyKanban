MyKanban

MyKanban is an open source project to create simple, general purpose Kanban software.

The BIG Idea …

Create an integrated application framework for task management, based on the Kanban board concept, that developers can use to quickly create sophisticated task tracking applications that will scale to enterprise levels Based on open source software so as to be accessible to all developers and organizations, and to gain access to the open source community to leverage high caliber free development resources

Provide clear and complete documentation and test cases to make the framework understandable and extendable by experienced developers within enterprises (e.g. Fenwick & West) and the open source community

MyKanban World View …

  • Boards … are collections of tasks that need to be viewed and managed as a group (e.g. my personal tasks, or all tasks for a given team)
  • Board Sets … are templates for collections of boards that share the same structure. The same project can be associated with more than one board in the same board set
  • Projects … are also collections of tasks that all support the same end deliverable (e.g. all tasks for a given client matter or IT project)
  • Tasks … are individual units of work assigned to one or more individuals
  • People … are associated with boards, projects, and tasks in a variety of ways: user, stakeholder, assignee, approver, etc.
  • Sprints … are time periods used for filtering views of tasks (e.g. 1/1/2015 through 1/15/2015) The full MyKanban model contains many more structures, but these are the most apparent

The 3 Layers of MyKanban

Sample UI #1 - the Object Browser Provides a window into the underlying API to help developers understand its internals. Can also be used as a basic administrative UI for boards, projects, tasks and users

Sample UI # 2 - Web Application

Provides an example of a simple, web-based Kanban board UI. This is a functional application that can be extended to meet particular needs

Sample UI #3 - Mobile

Provides an example of a Cordova-based mobile UI based on the MyKanban API. Can be extended as needed. Again, designed primarily as a learning tool, and not expected to meet all needs
